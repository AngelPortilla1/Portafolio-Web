# Angel Portilla - Portafolio Web

Un portafolio interactivo y moderno que muestra mis proyectos, habilidades y experiencia como Desarrollador Full Stack e Investigador en IA.

## 🚀 Características

- **Diseño Moderno:** Interfaz de usuario limpia, responsiva y con modo oscuro/claro.
- **Grafo de Tecnologías:** Visualización interactiva 2D de mis habilidades utilizando `react-force-graph-2d` y `d3`.
- **Sección de Proyectos:** Galería de proyectos con detalles, tecnologías utilizadas y enlaces a GitHub/Demo.
- **Rendimiento Optimizado:** Construido con Vite y React para una carga rápida y experiencia fluida.

## 🛠️ Tecnologías Utilizadas

- **Frontend:** React (v18), HTML5, CSS3, JavaScript (ES6+)
- **Estilos:** Tailwind CSS (v3.4)
- **Visualización:** react-force-graph-2d, d3
- **Iconos:** @heroicons/react
- **Build Tool:** Vite (v5)

## ⚙️ Instalación y Configuración Local

1. Clona el repositorio:
   ```bash
   git clone https://github.com/AngelPortilla1/Portafolio-Web.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd Portafolioweb2
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
5. Abre [http://localhost:5173](http://localhost:5173) en tu navegador para ver la aplicación.
